# Контрольна робота з JavaScript

## Опис
Цей репозиторій містить практичні завдання з тем:
- змінні та типи даних
- умови
- цикли
- функції
- масиви та об'єкти

## Завдання 1
Функція `summarizeNumbers(numbers)` рахує статистику масиву чисел:
- кількість елементів
- суму
- кількість парних чисел
- максимальне число
- категорію суми
